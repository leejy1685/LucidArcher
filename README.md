## 🧭프로젝트 목적
내일배움캠프의 강의에서 배운 것과 2D 기본제공 애셋을 메인으로 활용하여</br>
모바일 게임 '궁수의 전설' 을 모티브로 </br>
유니티 새 프로젝트에서부터 완성된 탑다운뷰 게임 만들어보기

## 📗게임 설명
_[ 프로젝트 명 - **루시드 아처** ]_

![Image](https://github.com/user-attachments/assets/d848c0cf-3cbb-4644-a21c-1629695474c3)

- 주인공은 "루시드 드리머"라고 불리는 특수한 존재로, 매일 밤 **다른 꿈의 세계**에 떨어진다.
- 꿈속은 몬스터가 들끓는 전장으로, 그 세계를 **정화하지 않으면 현실로 악몽이 침투**하게 된다.
- 현실과 꿈이 엮여 있어서, 플레이어가 꿈에서 얻은 능력이나 기억은 현실의 주인공 성장을 돕는다.
</br>

#### ◇ 자동 공격
모든 공격은 자동으로 이루어집니다. 움직임에 집중하세요!

#### ◇ 루시드 파워
좌하단의 루시드 파워 게이지는 공격 적중 시 차오르며 </br>
가득 차면 자동으로 게이지를 소모하며 일정 시간 공격속도가 2배 증가합니다.

#### ◇ 레벨 업과 스펙 강화
<img src="https://github.com/user-attachments/assets/d4b5beb3-2902-457f-9cae-f369ce6c1bae" width="50%"></br>
몬스터를 잡으면 노란 경험치 구슬이 떨어집니다. <br>
직접 줍거나 방을 클리어하면 자동으로 획득되며 화면 상단의 경험치 바가 차오릅니다. <br>
레벨 업 할 때마다 캐릭터를 강화시킬 수 있습니다.

#### ◇ 유체화
스태미나(좌상단)를 일정량 소모하여 무적 시간을 갖고<br>
적과 탄막을 통과할 수 있습니다.

- 자동 기본 공격
- 유체화

- 아이템 효과
- 넉백
